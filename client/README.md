# E-Commerce Client

Modern React-based frontend for the E-Commerce application.

## Technologies Used
- **React 19**
- **TypeScript**
- **Vite**
- **CSS** (Custom styling)

## Features
- **Product Listing**: Browse products with ease.
- **Filtering & Sorting**: Filter by category and stock availability. Sort by price and date.
- **Shopping Cart**: Real-time cart management with a slide-out drawer.
- **Responsive Design**: Optimized for various screen sizes.
- **Toast Notifications**: User feedback for actions.

## How to Run

1. Navigate to the client directory:
   ```bash
   cd client
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173` (default Vite port).
